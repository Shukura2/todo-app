# To-do App

To-do app is focused on activities management it also comes with expected date activities should be wrapped up

In this app, a user can:

- Add task
- See more
- Edit task
- Delete task
- Retain data after refresh

## How to install this project

- Clone the project
- Run npm install
- Run npm start
